简谱转换工具
============

作者：Tsukamotoshio
版本：0.1.3

将五线谱 PDF 批量转换为简谱 PDF，并可按需生成 MIDI 文件。
通过 Rich TUI 菜单界面进行操作，并内置简谱文本编辑器支持手动校对 OMR 识别结果。


使用方法
--------

1. 将五线谱文件（PDF / PNG / JPG）放入 Input 文件夹
2. 双击桌面快捷方式"简谱转换工具"，或运行安装目录下的 ConvertTool.exe
3. 用数字菜单选择功能：2 开始转换 / 4 打开简谱编辑器 / 6 查看帮助
4. 转换结果保存到 Output 文件夹

如果你从源码运行，请先执行：

```bash
pip install -r requirements.txt
```

如需使用 Homr 引擎，请提供本地 `omr_engine/homr` 克隆仓库，或安装 `homr` Python 包。
在 Windows 上，Homr 会自动检测 CUDA 或 DirectML，并优先使用 GPU 推理（如果可用）。

简谱文本编辑器
----------------

每次转换后，中间文件（.jianpu.txt）保存到 editor-workspace 文件夹。
手动校对流程：

1. 选择菜单项 4「打开简谱编辑器」
2. 从列表选择乐谱，记事本自动打开中间文件，同时打开原始图像供对照
3. 修改并保存简谱文本后关闭记事本
4. 选择「1. 开始生成简谱 PDF」即可输出校正后的版本


目录说明
--------

  Input\                    放入待转换的原始乐谱文件
  Output\                   保存转换后的简谱 PDF / MIDI
  editor-workspace\         存放供手动校对的简谱中间文件
  logs\                     运行日志（自动生成）
  THIRD_PARTY_NOTICES.md    第三方组件许可证说明


支持的输入格式
--------------

  .pdf          五线谱 PDF（推荐，支持多页）
  .png          五线谱图片
  .jpg / .jpeg  五线谱图片


已知缺陷与局限
--------------

- 识别准确率受乐谱质量影响：扫描模糊或排版复杂的乐谱可能出现错音、漏音
- 多声部支持有限：多声部乐谱转换结果可能只保留主旋律，部分音符丢失
- 不支持歌词输出：仅输出音符，不含歌词
- 处理速度较慢：Audiveris 启动耗时较长，多页 PDF 可能需要数分钟
- 调号 / 拍号边缘情况：少数非常规调号或变拍子乐谱可能不准确
