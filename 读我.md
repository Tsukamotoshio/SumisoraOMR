简谱转换工具
============

作者：Tsukamotoshio
版本：0.2.0-experimental

将五线谱 PDF 批量转换为简谱 PDF，并可按需生成 MIDI 文件。


使用方法
--------

1. 将五线谱文件（PDF / PNG / JPG）放入 Input 文件夹
2. 双击桌面快捷方式"简谱转换工具"，或运行安装目录下的 ConvertTool.exe
3. 按提示确认是否开始转换、是否生成 MIDI
4. 转换结果保存到 Output 文件夹


目录说明
--------

  Input\                    放入待转换的原始乐谱文件
  Output\                   保存转换后的简谱 PDF / MIDI
  logs\                     运行日志（自动生成）
  THIRD_PARTY_NOTICES.md    第三方组件许可证说明


支持的输入格式
--------------

  .pdf          五线谱 PDF（推荐，支持多页）
  .png          五线谱图片
  .jpg / .jpeg  五线谱图片


已知缺陷与局限
--------------

- 识别准确率受乐谱质量影响：扫描模糊或排版复杂的乐谱可能出现错音、漏音
- 多声部支持有限：多声部乐谱转换结果可能只保留主旋律，部分音符丢失
- 不支持歌词输出：仅输出音符，不含歌词
- 处理速度较慢：Audiveris 启动耗时较长，多页 PDF 可能需要数分钟
- 调号 / 拍号边缘情况：少数非常规调号或变拍子乐谱可能不准确


版本更新
--------

0.2.0-experimental（当前版本）
  - 首调唱名法（Movable-Do）：音符编号现在根据调号自动映射
  - 全新 Rich TUI 菜单界面
  - 新增 Oemer 深度学习 OMR 引擎（适合手机拍摄图片）
  - 内置简谱编辑器工作区，支持手动校对后重新生成 PDF
  - 权限错误修复：安装在 C:\Program Files 下以普通用户身份运行时不再闪退，
    改为显示友好提示，指引用户以管理员身份重新运行
  - MIDI 与 PDF 内容一致性修复：生成的 MIDI 现在与简谱 PDF 完全对应，
    均基于同一声部（首调旋律线），不再包含多声部或和弦
