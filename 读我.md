简谱转换工具
============

作者：Tsukamotoshio
版本：0.2.1

将五线谱 PDF 批量转换为简谱 PDF，并可按需生成 MIDI 文件。


使用方法
--------

1. 将五线谱文件（PDF / PNG / JPG）放入 Input 文件夹，或直接拖入 GUI 首页
2. 双击桌面快捷方式"简谱转换工具"，或运行 app.py / ConvertTool.exe
3. 在"文件 & 转换"页选择文件、引擎，点击"开始转换"
4. 转换结果保存到 Output 文件夹；识别出的 MusicXML 文件自动归档到 xml-scores 文件夹


目录说明
--------

  Input\                    放入待转换的原始乐谱文件
  Output\                   保存转换后的简谱 PDF / MIDI
  xml-scores\               识别后生成的 MusicXML 文件（可供移调引擎直接使用）
  editor-workspace\         简谱编辑工作区（.jianpu.txt / .ly / .pdf）
  logs\                     运行日志（自动生成）
  THIRD_PARTY_NOTICES.md    第三方组件许可证说明


支持的输入格式
--------------

  .pdf          五线谱 PDF（推荐，支持多页）
  .png          五线谱图片
  .jpg / .jpeg  五线谱图片


主要功能
--------

- 首页：文件选择、引擎选择（Audiveris / Oemer / 融合）、一键转换、实时进度日志
- 简谱编辑器：左侧图像预览（支持缩放、放大镜）+ 右侧文本编辑 + 一键导出 PDF
- 移调引擎：加载 MusicXML，选择目标调，左右对比预览，一键导出
- 关于：版本号、项目主页链接


已知缺陷与局限
--------------

- 识别准确率受乐谱质量影响：扫描模糊或排版复杂的乐谱可能出现错音、漏音
- 多声部支持有限：多声部乐谱转换结果可能只保留主旋律，部分音符丢失
- 不支持歌词输出：仅输出音符，不含歌词
- 处理速度较慢：Audiveris 启动耗时较长，多页 PDF 可能需要数分钟
- 调号 / 拍号边缘情况：少数非常规调号或变拍子乐谱可能不准确
- 导出 PDF 需要本地安装 LilyPond（已内置在分发包中）
